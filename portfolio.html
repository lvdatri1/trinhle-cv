Capability Building & Platform Ownership

Building an In-House Integration Platform Team

I led the uplift of Kāinga Ora's integration team from reliant on external vendors to full in-house ownership of a complex Enterprise Integration Platform spanning Azure Integration Services (Logic Apps, API Management, Service Bus, Event Grid) and Oracle Integration Cloud (OIC). The platform required intricate Infrastructure-as-Code configuration across Terraform, Bicep, and Azure DevOps CI/CD pipelines. Through pair-architecting, code reviews, and knowledge transfer, the team now independently manages, extends, and governs the entire integration estate — a capability that previously required ongoing external consultant support.
